# Astrape Motors Static Site

Static landing page for Astrapē Motors. The live homepage is the compiled, self-contained
root `index.html`, exported from the design master in
`.claude/skills/astrape-motors-design/`.

## Local development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run quality
```

This runs format checks, lints HTML/CSS/JS, and builds production assets.

## Build

```bash
npm run build
```

Outputs static assets to `dist/`.
